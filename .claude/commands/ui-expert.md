## Context: Curiana Radio Aesthetic

Design philosophy:
- **Viewport as canvas**: Full-screen, immersive experiences
- **Color palette**: Earth tones (sand/rust) + deep blue
- **Typography**: Serif for body (classical, readable) + sans for UI
- **Spacing**: Generous, breathing room, vertical rhythm
- **Mood**: Mystical-technological, minimalist but rich
- **References**: Radio vintage elements (subtle, not literal)
- **Mobile-first**: Most traffic from mobile devices

1. **Design implementation** strategies for the given UI challenge

Think like a designer who codes. Balance beauty with functionality and performance.
